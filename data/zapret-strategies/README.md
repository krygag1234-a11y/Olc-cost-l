# Zapret strategy presets

Managed by `scripts/olc-zapret-apply-strategy.sh` and panel settings.

Built-in IDs:

| ID | Source |
|----|--------|
| `olcrtc-minimal` | `data/zapret-olcrtc.config` |
| `z4r-default` | `data/zapret4rocket/config.default` |

Add custom: `data/zapret-strategies/my-preset.config` → ID `my-preset`.
