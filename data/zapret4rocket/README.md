# zapret4rocket assets (optional)

For `OLCRTC_ZAPRET_FULL=1`, place a zapret4rocket tree here (`config.default`, `lists/`, `fake_files.tar.gz`, …) or set:

```bash
export Z4R_SRC=/path/to/zapret4rocket
# optional:
export Z4R_REPO_URL=https://github.com/your-fork/zapret4rocket.git
```

Without `config.default`, `install-zapret-vps.sh` uses minimal `data/zapret-olcrtc.config`.
